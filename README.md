# assistencia-tecnica-adm
Sistema de Assistência Técnica desenvolvido para a disciplina de Engenharia de Software II, com foco no gerenciamento de clientes, equipamentos, ordens de serviço, funcionários e visitas técnicas. O projeto utiliza Python com FastAPI, seguindo boas práticas de desenvolvimento ágil com GitFlow, versionamento semântico e Conventional Commits.
